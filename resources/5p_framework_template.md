# 5P Persona Framework

| Pillar | Description |
| --- | --- |
| **Profile** | Demographics: Age, Job, Location, Income. |
| **Psychology** | Motivations, Fears, Goals. What keeps them up? |
| **Pains & Gains** | Current struggles vs. desired out-of-the-norm outcomes. |
| **Proficiency** | Tech-savviness. (e.g., "Sends Excel via Zalo") |
| **Principles** | Core values or rules they live/work by. |
